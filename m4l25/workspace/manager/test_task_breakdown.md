# 任务清单 - 智能日程助手

## 项目概览
- 目标：为 XiaoPaw 用户提供基于自然语言的日历事件管理能力（创建/查询/删除），集成至飞书聊天窗口
- 分工模式：流水线（PM → Dev → QA）
- 整体验收标准：
  - 自然语言解析准确率 ≥ 90%（基于10个测试用例）
  - 日历操作成功率 100%（创建/查询/删除）
  - 响应时间 ≤ 3秒
  - 上线时间：2周内

## 备注
- 所有 Dev 任务交付物需附单元测试（pytest），覆盖率 ≥ 80%
- QA 任务需复用 PM 提供的 10 条验收用例，不得新增主观用例
- 飞书 API 调用需使用沙盒预置的 mock_client，禁止直连生产环境
